# alx-listing-app-00

This repository contains a Next.js + TypeScript scaffold for the ALX listing project milestone 2. It includes:

- Layout components (Header, Footer, Layout)
- A hero section, filter pills, and a responsive listings grid
- TypeScript interfaces and a PROPERTYLISTINGSAMPLE constant

How to run locally:

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000
